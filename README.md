# STATE-OF-WASHINGTON-ELECTRIC-VEHICLE-POPULATION-DATA-ANALYSIS
STATE OF WASHINGTON ELECTRIC VEHICLE POPULATION DATA ANALYSIS

## PROJECT SUMMARY

The purpose of this project is to analyze the Washington Electric Vehicle (EV) dataset to uncover patterns in EV adoption, vehicle characteristics, pricing, and geographic distribution. This analysis includes univariate, bivariate, and multivariate visualizations to identify trends and relationships among features such as Model Year, Electric Range, Base MSRP, Make, EV Type, County, and CAFV eligibility.

The insights can support policy recommendations, consumer awareness, and strategic planning for automotive companies

## DATASET OVERVIEW

Columns: VIN (1-10), County, City, State, Postal Code, Model Year, Make, Model, Electric Vehicle Type, CAFV Eligibility, Electric Range, Base MSRP, Legislative District, DOL Vehicle ID, Vehicle Location, Electric Utility, 2020 Census Tract.

Columns and Rows: 264628 rows , 17 columns 

Data Quality: Minor missing values handled during preprocessing. Numerical values such as Base MSRP and Electric Range were checked for outliers and zero or missing values were replaced appropriately.
